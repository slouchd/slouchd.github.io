Today I Remember - The Cloud?!

2023-04-22

After trying to convince management, that the cloud is the right choice for the majority of our architectures moving forwards. I found myself trying to summarise why it was important. Although this sounds like an advert (haha). This does go towards encapsulating some of the benefits of cloud native.

Are you looking for a way to make your workloads more efficient and cost-effective? If so, you might want to consider adopting a cloud-native approach to your computing infrastructure. By offloading workloads to the cloud, you can save time, effort, and money, all while increasing the scalability and reliability of your systems.

One cloud platform that can help you achieve these benefits is Amazon Web Services (AWS). AWS offers a wide range of cloud-native services and tools that can help you build and manage your cloud infrastructure with ease. One such service is AWS Athena, which allows you to query data in different formats using the Presto query engine.

With Athena, you can easily analyze data stored in Amazon S3 buckets, as well as other data sources, such as relational databases or Apache Hadoop clusters. Athena supports a variety of data formats, including CSV, JSON, and Parquet, and allows you to query your data using standard SQL queries. And because Athena is serverless, you don’t need to worry about provisioning or managing any infrastructure yourself.

So if you’re looking for a way to make your data analysis more efficient and cost-effective, consider using AWS Athena with Presto. With its support for a variety of data formats and its serverless architecture, Athena can help you unlock the full potential of your data without breaking the bank.
